# VELA — Project Intelligence for Claude Code

## Brand
- Name: VELA — Gestão Inteligente
- Tagline: Inteligência que ilumina negócios
- Positioning: Premium AI-native SaaS for Brazilian aesthetic clinics

## Colors
- Obsidian (background): #0A0A0A
- Champagne (accent/CTA): #C9A96E
- Ivory (light bg): #F7F5F2
- Deep Teal (data/trust): #1A4A4A
- Amber Gold (text on light): #B8943C

## Typography
- Headlines: Cormorant Garamond (serif, elegant)
- UI/Body: DM Sans (clean, weight 200-400)
- Data/Metrics: DM Mono

## Tech Stack (never deviate from this)
- Framework: Next.js 14 App Router
- Language: TypeScript (strict)
- Styling: Tailwind CSS
- Database: PostgreSQL + Prisma ORM
- Auth: NextAuth.js
- AI Agents: OpenAI API

## Tone
- Direct, precise, confident, elegant
- Never: exclamation marks, generic language, emojis
- Always: data-first, result-oriented, premium feel

## MVP Modules (build in this order)
1. Landing page
2. Auth (login/signup)
3. Dashboard with KPIs
4. Agenda/scheduling
5. WhatsApp AI agent (Júlia)

## Rules
- Dark backgrounds by default
- Gold (#C9A96E) for CTAs and highlights only
- Never use blue, purple, or green
- All text in Portuguese (Brazil)
- Mobile-first responsive design
